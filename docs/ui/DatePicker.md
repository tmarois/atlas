# DatePicker

DatePicker is an input component for selecting dates.

```ts
import { DatePicker } from '@atlas/ui';
```

## Usage

```vue
<DatePicker v-model="date" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

See the [PrimeVue DatePicker API](https://primevue.org/datepicker/#api).
