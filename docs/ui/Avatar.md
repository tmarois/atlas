# Avatar

Avatar displays an image, icon, or initials representing a user or entity.

```ts
import { Avatar } from '@atlas/ui';
```

## Usage

```vue
<Avatar label="AB" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue Avatar API](https://primevue.org/avatar/#api).
