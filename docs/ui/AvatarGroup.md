# AvatarGroup

AvatarGroup stacks multiple Avatars together.

```ts
import { AvatarGroup, Avatar } from '@atlas/ui';
```

## Usage

```vue
<AvatarGroup>
  <Avatar label="AB" />
  <Avatar label="CD" />
</AvatarGroup>
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue AvatarGroup API](https://primevue.org/avatargroup/#api).
