# Topbar

Simple bar displayed above page content.

```ts
import { Topbar } from '@atlas/ui';
```

## Usage

```vue
<Topbar>
  <span>Announcements</span>
</Topbar>
```

## API

### Props
| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `pt` | `TopbarPassThroughOptions` | `undefined` | Passthrough options for internal elements. |
