# Textarea

Textarea is a multi-line text input with theming.

```ts
import { Textarea } from '@atlas/ui';
```

## Usage

```vue
<Textarea v-model="value" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue Textarea API](https://primevue.org/textarea/#api).
