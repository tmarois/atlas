# Tabs

Container for managing multiple tabbed panels.

```ts
import { Tabs, Tab, TabList, TabPanel, TabPanels } from '@atlas/ui';
```

## Usage

```vue
<Tabs value="0">
  <TabList>
    <Tab value="0">One</Tab>
    <Tab value="1">Two</Tab>
  </TabList>
  <TabPanels>
    <TabPanel value="0">Content 1</TabPanel>
    <TabPanel value="1">Content 2</TabPanel>
  </TabPanels>
</Tabs>
```
