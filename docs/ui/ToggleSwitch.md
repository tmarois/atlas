# ToggleSwitch

ToggleSwitch is a switch component for boolean values.

```ts
import { ToggleSwitch } from '@atlas/ui';
```

## Usage

```vue
<ToggleSwitch v-model="checked" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue ToggleSwitch API](https://primevue.org/toggleswitch/#api).
