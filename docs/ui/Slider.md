# Slider

Slider is an input component for selecting a numeric value on a range.

```ts
import { Slider } from '@atlas/ui';
```

## Usage

```vue
<Slider v-model="value" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

See the [PrimeVue Slider API](https://primevue.org/slider/#api).
