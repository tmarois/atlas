# InputNumber

InputNumber is an input field for numeric values with theming.

```ts
import { InputNumber } from '@atlas/ui';
```

## Usage

```vue
<InputNumber v-model="value" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue InputNumber API](https://primevue.org/inputnumber/#api).
