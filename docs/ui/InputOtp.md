# InputOtp

InputOtp is an input component for entering one-time passwords.

```ts
import { InputOtp } from '@atlas/ui';
```

## Usage

```vue
<InputOtp v-model="code" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue InputOtp API](https://primevue.org/inputotp/#api).
