# ToggleButton

ToggleButton is a two-state button component.

```ts
import { ToggleButton } from '@atlas/ui';
```

## Usage

```vue
<ToggleButton v-model="checked" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue ToggleButton API](https://primevue.org/togglebutton/#api).
