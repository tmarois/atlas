# AutoComplete

AutoComplete is an input with dynamic suggestions.

```ts
import { AutoComplete } from '@atlas/ui';
```

## Usage

```vue
<AutoComplete v-model="value" :suggestions="items" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

See the [PrimeVue AutoComplete API](https://primevue.org/autocomplete/#api).
