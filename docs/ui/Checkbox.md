# Checkbox

Checkbox is an extension to standard checkbox element with theming.

```ts
import { Checkbox } from '@atlas/ui';
```

## Usage

```vue
<Checkbox v-model="checked" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue Checkbox API](https://primevue.org/checkbox/#api).
