# InputText

InputText is an extension to standard input element with theming.

```ts
import { InputText } from '@atlas/ui';
```

## Usage

```vue
<InputText v-model="value" />
```

## API

### Props

None.

### Slots

None.

### Events

None.

Refer to the [PrimeVue InputText API](https://primevue.org/inputtext/#api).
