# AppTopbar

Simple bar displayed above page content.

## Example
```ts
import { Topbar as AppTopbar } from '@atlas/ui';
```

```vue
<AppTopbar>
  <span>Announcements</span>
</AppTopbar>
```

## Props
- `pt?: object` – passthrough options for internal elements.

## Slots
- `default` – bar content.

## Events
- None
